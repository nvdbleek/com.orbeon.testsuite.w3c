8.1.a navindex and accesskey (non-normative) 3 Steven Ball 8.1.a navindex and accesskey (non-normative) This test case is non-normative and assumes that navindex attributes will be recognized and interpreted as described in section 4.3.1 of the specification. Navigation order must be the Name input first, the Item input second, and the Quantity input third. Keyboard users can use the Tab key to test the navigation order. The input controls are also set to use access keys.
